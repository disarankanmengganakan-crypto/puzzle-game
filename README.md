# Arcana Clash (Three.js カードゲーム)

X投稿の雰囲気を参考にした、**HTML/CSS/JavaScript + Three.js** のブラウザ向け3Dカードゲームです。

## 特徴
- カードバトル形式（手札をクリックして攻撃）
- Three.jsで3Dボード/敵/手札演出を描画
- カード画像は外部素材を使わず、**Canvasで毎回自動生成**
- 1ターン1ドロー、マナ消費、敵ターン処理あり

## 起動
`index.html` をブラウザで開くだけで遊べます（ビルド不要）。

## 操作
- **カードをクリック**: プレイ
- **カードを引く**: ターン中1回のみ
- **ターン終了**: 敵の攻撃後、次ターン開始
- **リスタート**: 初期化
