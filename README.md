# ねこスライドパズル

ブラウザで遊べる 4x4 のスライドパズルです。

## 実行方法

`index.html` をブラウザで開くだけで遊べます。

## Image-gen アセット

本プロトタイプでは、Image-gen を想定したアセット運用として以下を同梱しています。

- `assets/cat-hero.svg`: キャラクター立ち絵
- `assets/puzzle-bg.svg`: UI パネル背景

### 推奨 Image-gen プロンプト例

- キャラクター: 「かわいい冒険猫、ゲームUI用、正方形アイコン、鮮やかな配色、フラットデザイン」
- 背景: 「パズルゲームの幻想的な背景、青紫グラデーション、やわらかい発光、ミニマル」

必要であれば同じファイル名で PNG を差し替えるだけで利用できます。
