# モンスター娘ローグライク（ブラウザ版）

- `index.html` をブラウザで開くとプレイできます。
- 地下1F〜30Fを実装したターン制ローグライクです。
- 入力: 矢印キー / WASD で移動、Spaceで待機。

## 注意
この実装は、提示画像の雰囲気に着想を得た**オリジナル実装**です。著作権上の理由から画像の完全な複製や既存IPのアセット流用は行っていません。


## 見た目の実装
- 主人公/敵はCanvas上で描画する簡易ちびキャラスプライトを追加しました。
- 環境制約のため外部画像生成APIは使わず、ゲーム内で自動生成する方式にしています。

- 種族ごとのシルエット差（角・耳・羽・尻尾）を追加。
- 待機アニメ（上下バウンス）と被弾アニメ（ヒット時の揺れ/赤フラッシュ）を追加。
- 右UIに立ち絵パネルを追加。
