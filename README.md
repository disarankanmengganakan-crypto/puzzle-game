# Arcana Clash (Three.js カードゲーム)

指定のX投稿イメージに寄せて、まず**アセットを自動生成**し、その生成物を使って対戦を行う3Dカードゲームとして再構築しました。

## 実装ポイント
- HTML/CSS/JavaScript + Three.js で構築
- カード画像・ボードテクスチャ・敵ポートレートをすべて Canvas で生成
- 生成したカードアセットのプレビューをHUD内に表示
- カードプレイ、マナ、ドロー、敵ターン、勝敗のゲームループ実装

## 起動
`index.html` をブラウザで開くとプレイできます。

## 操作
- 手札カードをクリック: カードを使用
- カードを引く: ターン中1回
- ターン終了: 敵の攻撃と次ターン開始
- リスタート: 全体を再生成して初期化
